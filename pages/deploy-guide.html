Deployment Guide

1 Push to GitHub

Initialize Git and push to GitHub:

git init

git add .

git commit -m "Initial commit"

git remote add origin your-repo-url

git push -u origin main

Make sure your repository is public or you have proper access configured.

2 Connect to Netlify

  1. Go to Netlify.com and sign up/login
  2. Click "New site from Git"
  3. Choose GitHub as your Git provider
  4. Select your repository

3 Configure Build Settings

Build Settings:

  • Build command: (leave empty)
  • Publish directory: /

4 Set Up Custom Domain

1. Purchase your .ai or .io domain

2. In Netlify, go to Site settings > Domain management

3. Click "Add custom domain"

4. Enter your domain and follow the DNS configuration steps

DNS Configuration:

Type: A Record

Name: @

Value: Netlify's IP

TTL: 3600

5 Final Configuration

Before Going Live:

  • Replace 'demo_client_id' with your Google OAuth client ID
  • Enable HTTPS (automatic with Netlify)
  • Test all functionality
  • Set up environment variables if needed

You're Ready to Go Live!

Your social media manager will be available at your custom domain in 24-48 hours.
